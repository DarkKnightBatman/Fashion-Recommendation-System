# Fashion Recommendation System 👗

This project is a content-based fashion recommendation system using deep learning.

## 🚀 Features
- Uses ResNet50 for feature extraction
- Computes similarity using cosine distance
- Recommends visually similar fashion items

## 🛠️ Tech Stack
- TensorFlow
- Scikit-learn
- NumPy
- OpenCV

## 📸 How it Works
1. Extract features from images using ResNet50
2. Store embeddings using Pickle
3. Use KNN to find similar images
